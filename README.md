# Assembly Metrics Visualization

Visualization based on the calculation result of [Assembly Metrics Toolkit](https://github.com/NAL-i5K/Assembly_Metrics_Toolkit).

## Installation

`npm install`

## Start HTTP server

`npm run start`

## Beautify code base

`npm run format`
